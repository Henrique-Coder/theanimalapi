- **Interactive documentation**: https://theanimalapi.pythonanywhere.com
- **Discord bot invitation url**: https://discord.com/api/oauth2/authorize?client_id=1109094043423608853&permissions=2048&scope=bot%20applications.commands

---

**Base API URL**: https://theanimalapi.pythonanywhere.com/api/v1/search/animal

#### Parameters:
- **name** - specify the name of the animal;
- **id** - specify ID of some specific or random animal;
- **lang** - choose the language of the animal returned in the api.
